# Segment Anything 2 Demo

这是一个基于 Vue 3 + Vite 构建的 Segment Anything 2 演示项目。

## 项目设置

### 安装依赖
```bash
npm install
# 或
yarn
```

### 开发环境运行
```bash
npm run dev
# 或
yarn dev
```

### 生产环境构建
```bash
npm run build
# 或
yarn build
```

## 项目结构
- `src/` - 源代码目录
- `public/` - 静态资源目录
- `src/views/` - 页面组件
- `src/components/` - 可复用组件

## 功能特性
- 响应式设计
- 视频展示
- 交互式演示
